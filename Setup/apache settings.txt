Add the following to the httpd.conf file in order to add an alias to access files not in the root web directory.


Alias /video /Videos
<Directory "/Videos">
    Options Indexes FollowSymLinks Includes ExecCGI
    AllowOverride All
    Require all granted
</Directory>
